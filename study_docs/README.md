# Study Docs Reading Map

This folder contains detailed learning notes for the project. For grading, the
shortest useful path is:

1. `../SUBMISSION_SUMMARY.md`
2. `../DESIGN_DECISIONS_AND_COMPARISONS.md`
3. `01_项目全景与最终目标.md`
4. `02_数据源与本地数据快照.md`
5. `04_图是怎么构建出来的.md`
6. `05_图分析层到底在回答什么问题.md`
7. `07_LLM新闻影响分析层.md`
8. `08_测试_验证与可信度.md`
9. `11_设计决策与方案比较.md`

The remaining files are optional learning aids:

- `03_新闻处理层是怎么工作的.md` explains the intermediate news tables.
- `06_交互层为什么分成CLI和网页.md` explains the CLI and web interfaces.
- `09_如何演示这个期末项目.md` is for preparing a walkthrough.
- `10_如何把这个项目讲成你自己的能力.md` is for personal interview practice.
